# API reference

> **Status: planned — not yet built.**

The **.NET 10 Blazor business API** is part of the jonnxor.is architecture but
has not been implemented yet. The current site is served entirely by Astro,
with content sourced from a local Directus instance via committed snapshots.

Once the API exists, this is where its reference will live:

- **Endpoints** — routes, request/response shapes, status codes.
- **Services** — the business-logic services behind the endpoints.
- **Authentication** — how callers authenticate and what's authorized.

Until then, there is nothing to document here. Check back after the API ships.
