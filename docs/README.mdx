# Welcome to the jonnxor.is docs

Guides, notes, and references for **jonnxor.is** — the personal corner of the web
belonging to Jón Agnar Stefánsson ("JonnXor"), a full-stack developer in Reykjavík.

The site is part portfolio, part saga: a place that gathers a body of work, long-form
writing on code, myth, and games, a hall of small games, a CV, and a living grimoire of
notes — all wrapped in a hand-built, Norse-inflected design system with three moods
(`dawn`, `rune`, and `neon`) and three tongues (Icelandic, English, and a little
Japanese, for fun). This is where the craft behind that site is written down: how it is
built, what changes between releases, and how the pieces fit together.

## What's here

- **[Changelog](./CHANGELOG.mdx)** — a running, human-readable record of what changed and
  when, newest first.
- **[Release notes](./release/)** — the longer story behind notable releases: what
  shipped, why it mattered, and anything worth knowing before you upgrade.
- **API reference** — *forthcoming.* Once the .NET 10 API lands, its endpoints and
  contracts will be documented under [`./api-reference/`](./api-reference/).

New here? The project [README](../README.md) is the technical front door — architecture,
stack, and how to run the site locally. These docs pick up where it leaves off.
